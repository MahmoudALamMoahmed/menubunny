

# تكبير صورة الغلاف على الموبايل

## المشكلة

ارتفاع الغلاف على الموبايل حالياً `h-40` (160px فقط)، وهو صغير جداً. الصورة الداخلية تستخدم `object-contain` + `max-h-full` فتتقلص بشكل ملحوظ داخل هذا الارتفاع المحدود.

## الحل

تعديل ارتفاع الغلاف في `src/pages/Restaurant.tsx` ليكون أكبر على الموبايل:

| الشاشة | قبل | بعد |
|--------|-----|-----|
| موبايل (default) | h-40 (160px) | h-56 (224px) |
| sm (640px+) | h-52 (208px) | h-64 (256px) |
| md (768px+) | h-80 (320px) | h-80 (بدون تغيير) |
| lg (1024px+) | h-96 (384px) | h-96 (بدون تغيير) |

## التغييرات

### 1. Restaurant.tsx - الغلاف الفعلي (سطر 373)
تغيير الكلاسات من:
```
h-40 sm:h-52 md:h-80 lg:h-96
```
الى:
```
h-56 sm:h-64 md:h-80 lg:h-96
```

### 2. Restaurant.tsx - Skeleton الغلاف (سطر ~297)
نفس التغيير على Skeleton Loading ليتطابق مع الارتفاع الفعلي:
```
h-56 sm:h-64 md:h-80 lg:h-96
```

هذا يضمن تطابق Skeleton مع الشكل النهائي (CLS = 0).
