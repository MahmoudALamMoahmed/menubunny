
# تحسين صفحة الطلبات وعرض البيانات الكاملة

## المشكلة الحالية

بعد فحص البيانات الفعلية في قاعدة البيانات، وجدت أن الطلبات تحتوي على معلومات غنية لا تُعرض:

### بيانات موجودة في قاعدة البيانات لكن لا تظهر:
1. **الحجم المختار** (`size`) - مثلاً: "كبير" بسعر 70 جنيه -- لا يظهر نهائياً
2. **الإضافات** (`extras`) - لا تظهر نهائياً
3. **اسم الفرع واسم المنطقة** - مدفونة داخل حقل `notes` كنص ("المنطقة: شارع الجنينه - الفرع: فرع المنيا") بدون تنسيق
4. **سعر التوصيل** - مضاف للإجمالي لكن لا يظهر كسطر منفصل
5. **طريقة الدفع** - لا تظهر إذا كانت "cash" (الدفع عند الاستلام)
6. **الملاحظات** - لا تظهر إذا كان العنوان موجوداً

---

## الحل

### 1. تحديث `OrderItem` interface ليشمل البيانات الكاملة

```typescript
interface OrderItem {
  id: string;
  name: string;
  price: number;
  quantity: number;
  total: number;
  size?: { id: string; name: string; price: number };
  extras?: { id: string; name: string; price: number }[];
}
```

### 2. تحسين عرض كل صنف في الطلب

بدلاً من عرض اسم الصنف فقط، يظهر:
- اسم الصنف + الحجم (إذا وجد) مثلاً: "فتة شاورما - كبير"
- قائمة الإضافات تحت الصنف بلون مختلف
- السعر الكامل (حجم + إضافات) x الكمية

### 3. عرض معلومات الفرع والمنطقة بشكل واضح

استخراج اسم الفرع والمنطقة من حقل `notes` وعرضهما بشكل منسق مع أيقونات، أو عرض `notes` كاملاً بتنسيق واضح.

### 4. عرض طريقة الدفع دائماً (حتى "cash")

### 5. عرض الملاحظات والعنوان معاً (وليس أحدهما فقط)

### 6. تحسين التصميم العام

- بطاقات أوضح مع فصل أفضل بين الأقسام
- عرض رقم الطلب واضح
- ألوان أفضل للحالات
- عرض سعر التوصيل كسطر منفصل (محسوب من: الإجمالي الكلي - مجموع الأصناف)

## تفاصيل التحسين البصري

### بطاقة الصنف الجديدة:
```text
+------------------------------------------+
| فتة شاورما - كبير              x1        |
| + جبنة موزاريلا، صلصة حارة              |
|                            70 جنيه       |
+------------------------------------------+
```

### قسم معلومات الطلب الجديد:
```text
معلومات العميل:
  الاسم: أحمد محمد
  الهاتف: 01012345678
  العنوان: شارع الجنينة

معلومات التوصيل:
  الفرع: فرع المنيا
  المنطقة: شارع الجنينة
  طريقة الدفع: الدفع عند الاستلام

تفاصيل الطلب:
  فتة شاورما - كبير x1 ............ 70 جنيه
  ------------------------------------------
  مجموع الأصناف:              70 جنيه
  سعر التوصيل:                30 جنيه
  ------------------------------------------
  الإجمالي:                  100 جنيه
```
